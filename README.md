# Breast Cancer Diagnosis using Machine Learning
## Researcher: Elaheh Heidarian Gorji

### Project Overview
This project focuses on the classification of breast cancer tumors using the Wisconsin Diagnostic Breast Cancer (WDBC) dataset. It implements various machine learning algorithms to distinguish between malignant and benign cases.

### Key Objectives
- Implementing **SVM** and **Random Forest** for binary classification.
- Feature extraction and dimensionality reduction for genomic and clinical data.
- Achieving high accuracy and sensitivity for clinical decision-making.

### Tools
- **Language:** Python
- **Libraries:** Scikit-learn, Pandas, Matplotlib, Seaborn
