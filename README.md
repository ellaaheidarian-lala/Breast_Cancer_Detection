# Comparative Analysis of ML Algorithms for Breast Cancer Diagnosis
## Lead Researcher: Elaheh Heidarian Gorji | Shahid Beheshti University

This research project explores the efficacy of supervised learning models in classifying breast cancer tumors. By utilizing the Wisconsin Diagnostic Breast Cancer (WDBC) dataset, I have developed a computational framework to analyze 30 distinct features (e.g., mean radius, texture, and area) to optimize diagnostic accuracy.
